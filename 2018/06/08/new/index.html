Markdown 基础语法

Jun 8, 2018

这是一个普通文本

当在句子的下一行加上==

或者加上–

就成了最高阶标题和第二阶标题
任何数量的=或-都有效，但至少是两个
在句子前面加入不同数量的#号的话，从一个到六个，分别可以创建一级到六级标题。H1~H6

#加一个

##加两个

###加三个

####加四个

#####加五个

######加六个

#######加七个，六个已经是极限了，所以这个仍然是六级标题

在句子前面打>号，是blockquote，引用一个段落

一个没有闪光点的人，身处在一个广阔的世界里，总是面临着声张自我与逃避审视目光的心理斗争。我在这里，请看我，请发现我。我不存在，别看我，别人比我有趣。
bq可以嵌套
一个没有闪光点的人，

身处在一个广阔的世界里，
总是面临着声张自我与逃避审视目光的心理斗争。

我在这里，请看我，请发现我。

我不存在，别看我，别人比我有趣。

只要在前面加上不同数量的>
区块内可以使用其他语法，包括标题，列表等

关于列表

在词语前面加上+号或者-号或者*号，注意和词语之间要加空格

  • 这是一个列
  • 这也是一个列
  • 这也是的
    上面的都是无序号列表
    有序号的列表是数字接上一个英文的句点
  1. sdqd1
  2. dsaf
  3. sag

在行首出现数字加句点的情况，为避免混乱，数字和点之间加反斜杠
18.22.22

用三个以上的星号、底线来建立一个分隔线，行内不能有其他东西

链接

  • 行内链接用方括号[]
    • 比如说baidu.com
      写法是[方括号里面是链接的名称] (后面紧跟着圆括号输入链接地址)
  • 参考链接
    • 链接文字的括号后面再接上另一个方括号，而在第二个方括号里面要填入用以辨识链接的标记，接着，在文件的任意处，把这个标记的链接内容定义出来：
    • [链接的名称] [链接的类型标识]
    • [链接类型标识] ：<http：//blablablabla>

强调

用号或_包围需要强调的语句
两边都是单个星号单个星号强调就是斜体
两边都是两个星号双星号强调就是加粗
如果三个呢**强调***就是斜体加粗
如果需要在里面插入星号或底线，在前面加反斜杠转译
如果两边都有空白

图片

一个惊叹号 !
接着一个方括号，里面放上图片的替代文字
接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上 选择性的 ‘title’ 文字。

! [ Alt text] (/path/to/img.jpg “Optional title”)

自动链接

http://baidu.com
< http://baidu.com >
用<>括起来

字体设置

<font face=”微软雅黑 color=gray size=42>文字

  • < font face =”微软雅黑 color=gray size=42>文字 *
